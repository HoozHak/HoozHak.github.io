Are You Wearing Your eProphylactic?


Web security is an important thing, especially now that EVERYTHING can be done online these days. The average person doesn't think about how easy it is for a web designer to gather your information as you are reading this. There is something called "navigator" that can be used inside of a website's coding. The navigator object is a property of the window object and can contain a slew of information about YOUR browser by using a list of properties.
Here are the properties of the navigator object and the code to access it.

Property appCodeName appName appVersion cookieEnabled geolocation language onLine platform product userAgent
Description Returns browser code name Returns browser name Returns browser version Returns true if browser cookies are enabled Returns a geolocation object for the user's location Returns browser language Returns true if the browser is online Returns browser platform Returns browser engine name Returns browser user-agent header

That alone should catch your attention at least a little bit. Here are a few links to websites that can help you browse the web in a safer way.

RicksCloud - This site by Rick's Cloud gives 12 great tips on how to go about your daily life on the internet.
Kasperky - Kaspersky of course does more than give you great antivirus software. This link will take you to a great top ten list of browsing the internet safely.
Finance.gov This website not only gives you tips for safe browsing practices, but also info about the importance of safe passwords, how to identify phishing, and more.
